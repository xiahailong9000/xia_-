位运算
 
位运算是直接对整数在内存中的二进制位进行的操作.
 
与：A&B；//-----------------且-&&---
或：A|B；//------------------或-----
取反：~A: //matlab中是逻辑非
异或：A^B; //matlab中表示幂运算
左移：A<<B; //----------------将A左移B位
右移：A>>B;
无符号右移：A>>>B; //java语法
 
与：1，指定位清零：void clr(int& a,int i){a&=~(1<<i);}
      2, 获取指定位的值: int get(int a,int t){return a&(1<<i);}
      3,保留某些位不变  a=a&oxFF;
      4,判断一个数是否是2的幂：bool isPow2(int a){return (a&(a-1))==0&&a!=0;}
      5,奇偶性判断： bool isOdd(int a){return a&1==1;}
 
或：1设置指定位的值：void set(int& a,int i){a|=(1<<i);}
      2,相反数： int opposite(int a){return ~a+1;}
 
异或：1交换两个数 void swap(int& a,int& b){a=a^b;b=a^b;a=a^b;}//不通过第三个数交换值
        2:指定位取反 void reverse(int& a,int i){a^=(1<<i)};
移位：1，2的次幂：a<<2   //--将a扩大4倍
         2，循环移位  void rol(int& a,int k){a=a<<k|a>>32-k};  //--不丢失信息的移位--将移出去的数据在放到高位
 
位运算的简单应用
average函数//平均值-----int average =(x&y)+((x^y)>>1);
abs函数//绝对值------int abs(int x){int y=x>>31; return (x+y)^y;}
max函数//两者中较大值-------int max(int x,int y){return x^((x^y)&((x-y)>>31));}
